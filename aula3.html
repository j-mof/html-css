Código-fonte e Pré-formatação

O Comando document.getElementeById('teste') é escrito em linguagem Javascript.

O comando CODE cria uma fonte monoespaçada. Ela é ótima para ler códigos.

O CODE não tem indentação. Para ter indentação é necessário envelopar o comando dentro da tag PRE (pré-formatação).

Citação deslocada - BLOCKQUOTE

Usando a tag blockquote a citação do livro ou texto aparecerá deslocada. Nessa citação também usei o itálico e as aspas de citação.

Dentro do BLOCKQUOTE eu posso concatenar a tag CITE = "copiar o link do livro".

A tag CITE não aparecerá visualmente na página, mas entrará no sistema de buscas do Google (referenciamento).
